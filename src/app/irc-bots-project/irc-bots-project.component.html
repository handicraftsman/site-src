I've written multiple IRC bots in various languages. My latest (and one of most stable) bot is ParticleBot2.

ParticleBot2

ParticleBot2 is written in C++ and supports serving channels in multiple IRC networks and extending using C++ plugins.

ParticleBot2 depends on Guosh, ParticleDI, ParticleINI, SQLite3 and CURL.

Optional dependencies include libxml++ and jsoncpp.
