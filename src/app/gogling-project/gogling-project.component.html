Gogling

Gogling is an application server with Lua API I wrote back in 2016 and remade in August 2018. Simple example: 

{{ example1 }}

This will reply to your requests to / with Hello, World!.

Current version is a full rewrite, but you still can access old one at development and updpack-into-package branches

Gogling supports importing prebuilt (as plugins) Golang's standard library. To build it, you must have ruby, bash and go. 

{{ example2 }}

Now you can find built standard library as a set of plugins. (warning: they take ~860 MB on your disk so you shouldn't push all of them to the server)
