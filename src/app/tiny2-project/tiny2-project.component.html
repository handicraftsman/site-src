tiny2-object

tiny2-object is a partial OOP implementation in C.

It supports method tables, virtual method tables, and inheritance.

It DOES NOT support such features as data encapsulation (private/protected fields; but you can create a class with a pointer to a private object) and multiple inheritance.

tiny2-containers is a container implementation for tiny2-object.

It has string wrappers (for storing strigs) lists, vectors, queues and maps.
